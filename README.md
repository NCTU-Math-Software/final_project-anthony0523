輸入一個x∈(1,27.5]，此方程式將輸出一個N∈正整數並滿足S(N+1)>x>sN (如果x>27.5，計算時間將超出10分鐘)
For example:

INPUT:
    x=5
OUTPUT:
    S82=
    4.990020079909081

    S83=
    5.002068272680166

    S83 > 5 > S82, so N=82
